== Description ==

yTicket for WP allows you to connect your website to your platform help desk yTicket.<br />
yTicket for WP is a plugin that allows you to interface your own website created with the WP platform for Help Desk Ticketing System yTicket.<br />
With this plugin, you can:<ul>
<li>1) create a form with which customers can open a ticket from a page of the website</li>
<li>2) create a form with which customers can check the processing status of a ticket</li>
</ul>


== Requirements ==

Instance of the platform ticketingsystem.it


== Installation ==

Upload the yTicket plugin to your blog with ftp client or Wordpress Admin area;<br /> 
Activate it;<br />
Enter your yTicket url, username and password generated from your yTicket platform.<br />
Insert into post or page this code:<br />
- {yticket} to generate the form for opening a new ticket<br />
- {yticket-check} to generate the form for checking the status of a ticket processing<br />
<br />
Good Trouble Ticket with yTicket :)<br />
